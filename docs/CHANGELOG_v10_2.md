# Ripple Infrastructure Twin v10.2 — Cinematic UX Fix

Cambios:

- Botón de cinemática ahora abre una experiencia a pantalla completa real.
- Escenas con avance automático, anterior/siguiente, pausa, cerrar y teclas Escape/flechas/espacio.
- Animaciones visuales: nodos FIAT/RLUSD/XRPL/XRP/ORDERBOOKS, partículas de flujo y barra de progreso.
- Correcciones de interfaz para evitar textos cortados en tarjetas, métricas y wallets.
- Rejillas responsivas: los recuadros crecen según contenido.
- Billeteras separadas por activo/red con botón de copiar.
- Añadida tarjeta para XRP nativo en XRPL usando la misma cuenta XRPL.

Nota: los navegadores pueden bloquear autoplay con sonido. La web intenta reproducir desde el inicio, y si el navegador lo bloquea, se activa con el primer clic o con “Entrar con música”.
